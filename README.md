# Bank & Account Management System

A simple **Bank Management System** built using **Spring Boot, MySQL, HTML, and CSS**.  
This application allows users to manage banks and customer accounts with basic CRUD operations.

## Project Overview

The Bank & Account Management System provides functionality to add bank details, create customer accounts, and display account information including account holder name and balance.

## Features

### Bank Management
- Add new bank details
- Store bank name and IFSC code
- View available banks

### Account Management
- Create new customer accounts
- Select bank while creating an account
- Store account holder details
- Maintain account balance
- View account details

### Data Operations
- Add Records
- Display Records
- Update Records
- Delete Records

## Technologies Used

### Backend

- Spring Boot
- Spring Data JPA
- Hibernate

### Frontend
- HTML
- CSS
